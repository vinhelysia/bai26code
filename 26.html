Hãy bấm vào các từ có nét đứt để xem cách tạo và gọi Hàm.

# 1. Định nghĩa Hàm KHÔNG có giá trị trả lại
def thong_bao(msg):
    print("Xin chào:", msg)
    return

# 2. Định nghĩa Hàm CÓ trả lại giá trị (Kiểm tra nguyên tố)
def prime(n):
    if n < 2:
        return False
    for k in range(2, n):
        if n % k == 0:
            return False
    return True

# 3. Lời gọi hàm (Chương trình chính)
ten = input("Nhập tên: ")
thong_bao(ten) # Gọi hàm tự định nghĩa

so_nhap = int(input("Nhập một số: "))
ket_qua = prime(so_nhap)
print("Kết quả nguyên tố:", ket_qua)

Bấm vào một cấu trúc (def, return...) ở đoạn code trên để xem giải thích chi tiết.
